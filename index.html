Motivation

This page contains material for teaching others and myself.

The aim is to make abstract concepts in the field of econometrics more tangible and thus to improve the understanding.

There are different ways to achieve this including:

  • Replicate and extend empirical studies.
  • Illustrate concepts using simulation studies.
  • Playing around with verbal interpretation, mathematical formula and code.

Maybe some of the content can also improve your understanding

Replication Material for Juselius (2006)

Find some replication material for:

Juselius, K. (2006). The cointegrated VAR model: methodology and applications. Oxford University Press.

Use the links below to get to the Bookdown page and the corresponding Github repository:

Bookdown Page

Github Repository

This is one of the best books about the empirical application of cointegrated VAR models in empirical macroeconomics. However, most of the procedures are implemented using the software package RATS and CATS.

The material provides some implementations using the free software package R. Some of the procedures can be replicated using the R package urca. However, these procedures have to be changed a bit to replicate the methods and the results of the book.
